## Run Locally

Clone the project

```bash
  git clone https://github.com/Priyanshu0007/breatheESG.git
```

Go to the project directory

```bash
  cd breatheESG
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
